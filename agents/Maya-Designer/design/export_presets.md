# Export Presets for Diet Calorie App v1

Purpose: Standardize PNG exports for handoff and review.

Screens to export (6 key screens):
1. Onboarding
2. Home / Daily Summary
3. Search Results
4. Food Detail
5. Meal Logging (sheet/modal)
6. Settings (Calorie target)

Canvas size:
- Mobile width: 375px (iPhone X reference)
- Height: variable; recommended 812px for full-screen context

Export settings:
- Format: PNG
- Scale: 1x (375x812) and 2x for high-density preview (750x1624)
- Slicing: export artboard as full PNG
- Naming convention: diet_v1_<screen>_1x.png, diet_v1_<screen>_2x.png

Example filenames:
- diet_v1_onboarding_1x.png
- diet_v1_home_1x.png

Location for handoff: upload PNGs to shared assets folder and paste links in the handoff message.

Note: I will create the wireframes in Figma and export these PNGs. If Figma access is restricted, provide a shared Figma team/project link.
